# example-1

Basic example showing how Featurevisor works.

## Initialize

```
$ mkdir example-1
$ cd example-1
$ npx @featurevisor/cli init
```

## Installation

```
$ npm install
```

## Usage

### Lint YAMLs

```
$ npm run lint
```

### Build datafiles

```
$ npm run build
```

### Test features

```
$ npm test
```

### Start local server

Generates and serves status site:

```
$ npm start
```
